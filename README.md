# Student Grade System

A simple **Python console application** to manage student records.  
This project allows you to **add, view, update, delete, and search students**, calculate total marks, percentage, and assign grades.

---

## Features

- Add a new student with:
  - Roll number
  - Name
  - Marks for Math, Science, and English
- View all students with:
  - Marks
  - Total marks
  - Percentage
  - Grade
- Update a student's name and marks
- Delete a student record
- Search a student by roll number
- Automatic grade calculation based on percentage:
